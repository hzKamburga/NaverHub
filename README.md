# Roblox Script Hub

A legal, educational Roblox Lua script hub with a modern GUI interface.

## Project Structure

```
RobloxHub/
├── main.lua              # Main hub loader
├── gui/
│   ├── interface.lua     # GUI framework
│   └── themes.lua        # Color themes and styling
├── scripts/
│   ├── example1.lua      # Example script 1
│   └── example2.lua      # Example script 2
└── utils/
    └── helpers.lua       # Utility functions
```

## Features

- Modern, glass-morphism inspired UI
- Easy to use interface
- Modular script system
- Customizable themes
- Smooth animations

This is for educational purposes only. Use responsibly within Roblox's Terms of Service.
